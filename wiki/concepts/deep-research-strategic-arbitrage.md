# Deep Research Strategic Arbitrage

Framework for monetizing a [[claude-code]] Max subscription by treating frontier AI access as an arbitrage opportunity: you pay a flat fee, the model's output is worth more per-hour than any human contractor you could hire at the same rate.

## Core thesis

Phase 3 autonomous agents (see [[phase-3-agents]]) have crossed the threshold where a single developer can ship products that previously required a team. The "strategic arbitrage" is the delta between the $200/mo Claude Max cost and the value of work it produces. Finn's version: use Claude Code + Cowork to run [[overlook-strategy]] at near-zero marginal labor cost.

## Key plays identified

- **[[productized-services]]** — package repeatable deliverables (brand audits, site builds) at fixed prices, let agents do 80% of the work
- **[[boring-business-automation]]** — target unsexy industries (HVAC, dental, legal) where even basic automation is novel
- **[[ai-agency-niches]]** — regulated industries (legal, healthcare, finance) pay premium for AI-augmented workflows because stakes are high and labor is expensive
- **MCP plugin marketplace** — build and sell integrations for the Claude ecosystem
- **Content + audience** — use AI to produce volume content, build audience, monetize via courses/consulting

## The "WAT" filter

Every idea should pass the [[wat-framework-marketing]] test: Would a potential customer say "wait, you can do that?" If yes, the value prop is clear enough to sell.

## Finn's position

Finn already has the stack ([[claude-code]], [[overlook-strategy]], [[overlook-portal-webapp]]) and the client pipeline. The arbitrage is executing faster and broader than a solo founder normally could. See [[claude-max-arbitrage]] for the cost math.
