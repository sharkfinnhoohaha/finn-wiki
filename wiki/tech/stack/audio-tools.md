# Audio tools

Finn produces music professionally and runs [[overlook-audio]] (hardware/firmware for audio gear). He's a [[berklee-online|Berklee Online]] student with `.edu` access. This is decision-aid for new tools, not a tutorial — Finn already knows DAWs.

## Pick (current setup)

- **DAW:** Logic Pro (one-time $200, M1 Mac). Industry standard at Berklee, no subscription, runs natively on Apple Silicon.
- **Samples:** Splice on the [[student-discounts|Berklee deal]] ($4/mo Sounds+ vs $12.99 retail) — half-off and includes a free Baby Audio plugin.
- **Plugin bundle:** Slate Digital All-Access Pass via [Berklee Online deals](https://online.berklee.edu/deals) — up to 70% off (140+ plugins).
- **Mastering for finals:** iZotope Ozone if you own it; otherwise LANDR or eMastered for quick passes.

## When NOT to use the defaults

- **Building a song to release commercially with budget for one master** → eMastered or LANDR for $5-20 beats hand-mastering for tight deadlines.
- **A track for a Berklee assignment** → hand-master with Logic stock + free plugins. The point is learning, not the AI shortcut.
- **A track that might get sync-licensed** ([[berklee-music-supervision-1]] context) → human mastering engineer. AI services can disqualify you for rights/clearance reasons in some sync contexts.

## Worth knowing about (free / cheap)

- **Valhalla Supermassive** — free reverb, ridiculous range. Install yesterday.
- **TDR Nova** — free dynamic EQ that beats most paid options.
- **Vital** — free wavetable synth, Serum-tier capability.
- **Spitfire LABS** — free orchestral instruments. Quality is shocking for the price.
- **Cymatics free packs** — they email free samples weekly. Subscribe, ignore the upsells.
- **Kilohearts Phase Plant + free Snapins** — modular synth + free utility plugins.
- **Moises** — AI stem separation. **25% off first year via Berklee** ([deals page](https://online.berklee.edu/deals)).

## Avoid / paid-default trap

- **Komplete one-time bundles** — used to be a great $99 student deal. **No longer offered as of 2026** — only Reaktor 6 has 50% off ($199 → $99). Confirm before buying.
- **Splice on retail subscription** when the [[student-discounts|Berklee deal]] cuts it 70%.
- **Buying every Slate plugin a la carte** when All-Access Pass via Berklee is ~$45/yr for everything.
- **Pro Tools subscription** for personal projects. Logic does 95% of what Pro Tools does for $200 once.

## Overlook Audio context

[[overlook-audio]] is hardware/firmware focused. The DAW work is about evaluating signal paths, A/B testing prototypes, recording demos. Logic + Splice + Slate covers all of this. Don't add a second DAW for [[overlook-audio]] needs unless there's a specific requirement.
