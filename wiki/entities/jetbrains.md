# JetBrains

Maker of IntelliJ, WebStorm, PyCharm, DataGrip, GoLand, and the rest of the JetBrains IDE family.

## Finn's status

✓ All Products Pack activated, **DataGrip + WebStorm downloaded** as of April 28, 2026.

> [!warning] Educational license is non-commercial only
> JetBrains' educational license is **strictly for non-commercial educational purposes**, including academic research. It cannot legally be used for **paid client work, freelance work, or any project with commercial intent** — regardless of whether Finn is the one being paid directly. This means using student WebStorm/DataGrip on [[overlook-strategy-positioning|Overlook Strategy]] client engagements technically violates the license.
>
> Practical guidance lives in [[student-discounts]] under "JetBrains heads-up" — the cleanest path is to keep student licenses for personal projects + [[school-berklee|Berklee coursework]] and use VS Code / Cursor for commercial client work. After graduation, JetBrains offers a **40% personal-license discount valid for 2 years**.

## What's included

- Every JetBrains IDE (IntelliJ Ultimate, WebStorm, PyCharm Pro, DataGrip, GoLand, RubyMine, PhpStorm, Rider, AppCode, CLion, RustRover, Aqua)
- AI Assistant (subject to current bundle terms — credit limits may apply)
- ReSharper, dotTrace, dotMemory, dotCover, dotPeek
- Toolbox App
- Renews annually with `.edu` re-verification

## What's excluded / restricted

- **Commercial use** — see warning above
- Team Tools licensing (TeamCity, YouTrack, Space Cloud team plans) — those are separately licensed
- Some AI Assistant credit limits may differ from paid commercial tier

## Where it fits in Finn's stack — for non-commercial work

- **WebStorm** > VS Code for large refactors and TS-heavy [[next-js-patterns|Next.js]] work on personal/learning projects
- **DataGrip** beats Postico/TablePlus for [[alembic-postgres-patterns|Postgres]] schema browsing
- **PyCharm Pro** for personal FastAPI / SQLAlchemy projects
- For client work: pivot to VS Code + Cursor + free DB tools (Postico Lite, TablePlus free tier)
