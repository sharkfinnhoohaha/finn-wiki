# Tailwind CSS

Utility-first CSS framework used across all of Finn's [[next-js-patterns]] projects. Default aesthetic: monochromatic (white bg, neutral-900 text, neutral-100/200 borders). Part of the core frontend stack alongside React and TypeScript on [[vercel]].
