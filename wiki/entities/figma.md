# Figma

Browser-native design tool. Used on every [[overlook-strategy-positioning|Overlook Strategy]] client engagement for layout exploration, design system tokens, and developer handoff before [[next-js-patterns|Next.js]] implementation.

## Student access

✓ **Finn's status:** activated.

- **Figma Education plan** is free for 2 years with `.edu` verification via SheerID, with renewal possible up to 4 years total.
- Includes Figma Professional features (unlimited Figma + FigJam files, advanced libraries, plugin development, dev mode for handoff).
- Apply at https://www.figma.com/education/.
- Some org-tier features and team-management capabilities are excluded vs. full commercial Professional plan; verify any team-feature dependency before relying on it for client work.
- Rolls back to free tier limits if not renewed.

## Where it fits

- Primary design surface for client work (logo concepts, layout, component spec).
- FigJam for [[overlook-strategy-positioning|brand strategy]] sessions and architecture sketches.
- Dev Mode + plugin export for handoff to Cursor / Claude Code workflows.
- Pairs with **Octicons** (free Figma library via the [[github-copilot|GitHub Student Pack]]) and **IconScout** (60 icons/mo via Pack).

## Notes

- Don't put client work in the personal `.edu` Figma if the client expects to own files — set up a client-owned team and invite the Edu account in.
- Education plan loses access to organization-level admin features that are rarely needed at this scale anyway.
