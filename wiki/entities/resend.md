# Resend

Transactional email service from the [[vercel]] ecosystem. Write emails as React components using the `react-email` library, ship them from your codebase.

Pro tier $20/month for 50K emails/month. Used in [[ai-newsroom-pipeline]] for transactional flows: subscription confirmations, password resets, comment notifications, editor approval summaries.

React Email components live in your app's monorepo, versioned alongside the code that sends them. No separate email template drag-and-drop UI to stay in sync with.
