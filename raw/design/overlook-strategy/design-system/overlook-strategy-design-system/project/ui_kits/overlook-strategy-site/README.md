# Overlook Strategy — Marketing Site UI Kit

A hi-fi recreation of the agency's own marketing site. Editorial, cream-on-ink hero, paper body. Based on the logo, business card, and the HUD/meta motifs lifted from the `three-altitudes` codebase (Vector / Engine Room stage).

## Structure
- `index.html` — assembled home page (hero → approach → selected work → contact)
- `Nav.jsx` — top navigation, fixed, paper background with hairline
- `Hero.jsx` — ink-inverse hero with grain, serif display headline
- `Approach.jsx` — three-column editorial block with hairline rules
- `Work.jsx` — selected work list, numbered, alternating ink/paper
- `Contact.jsx` — cream-on-ink footer with coordinates + links

## Notes
- All text is in-voice (see README brand voice).
- Icons are Lucide via CDN.
- Imagery is represented with labeled placeholder blocks ("Photo: Ventura coast, morning") since no brand photography was provided.
